# Cartão Digital — Pablo Santos

Esta pasta contém um cartão digital (PNG, PDF e HTML) pronto para publicar no GitHub Pages.

**Ficheiros gerados**
- `index.html` — página do cartão (referencia `assets/avatar.png` e `assets/qr.png`)
- `styles.css` — estilos
- `assets/avatar.png` — avatar gerado
- `assets/qr.png` — QR code (vCard)
- `pablo_santos_card.png` — imagem do cartão
- `pablo_santos_card.pdf` — PDF prontp para impressão

## Como publicar no GitHub Pages (passos rápidos)
1. Crie um repositório no GitHub (por exemplo `pablo-santos-card`).
2. Faça clone local, copie todos os ficheiros desta pasta para a raiz do repositório.
3. Commit e push para o `main` (ou `master`).
4. No GitHub, vá a **Settings > Pages** e defina a fonte para `main` branch (root) — o site ficará disponível em `https://<seu-username>.github.io/<repo>`.

## Notas
- O QR codifica um vCard com os seus dados; ao escanear, dispositivos compatíveis oferecem guardar o contacto.
- Se quiser substituir o avatar por uma foto real, substitua `assets/avatar.png` pelo ficheiro desejado mantendo o mesmo nome.
